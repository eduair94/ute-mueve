# @ute-mueve/openapi

OpenAPI 3.1 specification and example JSON fixtures for the UTE Mueve bridge.

- `openapi.yaml` — generated by `pnpm openapi:export` from the bridge runtime Zod schemas.
- `fixtures/*.json` — sanitized real responses captured via `pnpm capture`. Used as `example` values in the OpenAPI spec, and as contract-test inputs.

## Refreshing

```bash
UTE_UNIQUE_KEY=<hex> UTE_USER_ID=<your-userId> pnpm capture
pnpm types:generate     # regenerates quicktype companions in @ute-mueve/types/generated
pnpm openapi:export     # regenerates openapi.yaml from Zod schemas
```
